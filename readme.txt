== Description ==
Taskwizard is a WordPress-based automation platform for interacting with APIs and formatting data.

== Installation ==
1. Upload the `taskwizard` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
